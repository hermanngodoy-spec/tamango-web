# tamango-web

Landing page pública de **TAMANGO SpA** (tamangospa.cl) — insumos industriales y de aseo del **Grupo Chavilca**. Sitio estático: solo HTML y CSS, **sin build, sin backend**. Hermano de [chavilca.cl](https://chavilca.cl) y [amaruspa.cl](https://amaruspa.cl).

## Ángulo comercial
"Los insumos que el grupo usa todos los días en sus propias operaciones, disponibles para su empresa" — extrapolación del consumo interno hacia clientes, con stock local en la Provincia de Chañaral.

## Contenido
| Archivo | Qué es |
|---|---|
| `index.html` | Página completa: hero con el isotipo (cuero), catálogo (4 categorías), sección "Parte del Grupo Chavilca" (ventana Chavilca + hermanas con logos reales), contacto. |
| `favicon.svg` | El cuero/tamango. |
| `404.html`, `robots.txt`, `sitemap.xml` | Soporte y SEO (dominio tamangospa.cl). |

Contacto: contacto@tamangospa.cl

## Deploy
Cloudflare Pages conectado a este repo (framework preset: *None*, sin build). Cada push a `main` publica solo.
